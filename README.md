# TOEIC Master v9.1

## 本版完成
- 雙人線上對戰：建立房間、輸入房號、兩人同步答題、即時比分與勝負
- 隊伍功能：共同 XP 目標、隊員貢獻排行、邀請碼、隊伍進度
- 配對遊戲答錯倒扣 20 分（最低 0 分）
- 559 個單字全部補上詞性與例句欄位
- Google 登入、排行榜、投稿、管理員審核、收藏、錯題、文法詳解均保留

## 上線方式
1. 先刪除 GitHub Repository 內舊檔，再上傳此資料夾內全部檔案。
2. Firebase → Firestore Database → 規則，貼上 `firestore.rules` 並發布。
3. Google 登入使用正確的 Firebase Web App 設定。

## 線上對戰測試
- 兩個不同瀏覽器或手機登入不同 Google 帳號。
- 玩家 A 建立房間並分享房號。
- 玩家 B 輸入房號加入。
- A 按「開始對戰」。
